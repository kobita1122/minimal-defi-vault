# Minimal DeFi Vault (ERC-4626)

This repository contains a high-quality, professional implementation of the **ERC-4626 Tokenized Vault Standard**. It is designed for developers looking to build yield-aggregators, lending pools, or any system where assets are managed to earn interest.

## Key Concepts
- **Asset**: The underlying token (e.g., USDC, DAI, WETH) deposited into the vault.
- **Shares**: The yield-bearing tokens minted to the user representing their claim on the vault's assets.
- **ERC-4626**: A standard API for tokenized vaults that improves composability across DeFi protocols.

## Technical Details
- **Solidity 0.8.20**
- **OpenZeppelin 5.0**
- **ERC-20 Compatibility**: Shares are standard ERC-20 tokens.
